# MindTap Printer

MindTap Printer is an extension that allows you to extract text from Cengage MindTap ebook pages as you skim through.

## App Tutorial

1. Go to an ebook at Cengage MindTap.
2. Find a chapter you would like to print, and start skimming through the pages. MindTap Printer will automatically collect all of the content on the textbook
3. Copy the text by clicking on the extension icon, and paste it whereever you like (Ex. Google Docs) to print.
